# Librarything.com Extended search
A custom search for Librarything.com Book pages

## How to Use
Install Tampermonkey browser extension (Chrome, Firefox, Safari, Edge)

Add this script by copying the raw URL:

    https://raw.githubusercontent.com/myfta/librarything-search/main/librarything-ia-hathitrust-search.js

Open any LibraryThing book page (e.g., librarything.com/book/...)

The search toolbar will appear below the book title with two buttons:

📚 Internet Archive - searches by ISBN or title/author

🔍 HathiTrust - searches by ISBN or title/author

## Script Features

Priority order: ISBN → Title + Author → Title only

Smart extraction: Finds book metadata on the page automatically

Clean UI: Styled toolbar with hover effects

Error handling: Gracefully handles missing data

Responsive: Buttons wrap on narrow screens
